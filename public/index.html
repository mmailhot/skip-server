SKIP

Simple Kommunication Interface for Pebble

Super Alpha Edition

What is SKIP?

SKIP is an API that allows you to quickly send a message directly to your Pebble Smartwatch though Android and Google Cloud Messaging. Since is uses simple http POST requests, messages can be sent easily from a wide variety of languages (even bash scripts). All code is MIT Liscensed.

How does it work?

After you download and install the .apk it will issue you an API key that is linked to your device. Simply send a POST request to the endpoint (instructions below).

Caveats + Warnings

  • This was a quick weekend project - therefore security, stability, etc. are not quaranteed. Raise any issues on the github repo's linked below.

  • The endpoint is currently hosted on a free heroku instance which has limits on both throughput/speed and the number of devices that can be registered. If you are going to root/wipe/change phones please unregister the old one. Sorry, I'm a broke student and can't afford hosting for it.

  • Android 2.3+ Only.

Detailed instructions

  1. Download and install the APK from the link below
  2. Select the register device link
  3. Your API key should appear
  4. WARNING: If you unregister your device, when you reregister your API key will be different
  5. Send a POST request to 
    http://skipapebble.herokuapp.com/devices/YOUR_API_KEY
    with two parameters in the body title containing the title of the notification and body containing the body of the notification
  6. The API will return a JSON Object with a key success containg a true/false value

Examples

cURL

curl -d "title=YOUR TITLE HERE&body=YOUR BODY HERE" http://skipapebble.herokuapp.com/devices/YOUR API KEY
